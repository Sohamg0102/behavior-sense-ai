🧠 BehaviorSense AI

Multimodal Behavioral Analysis System (CV + Audio + NLP)

⸻

🚀 Project Overview

BehaviorSense AI is an intelligent system that analyzes human behavior using multiple modalities:
	•	🎥 Facial expressions (Computer Vision)
	•	🎙️ Voice signals (Audio Processing)
	•	📝 Text & speech (NLP)

The goal is to estimate:
	•	Stress level
	•	Confidence level
	•	Behavioral insights

⸻

🎯 Current Features

✅ Real-time webcam input
✅ Face detection using MediaPipe
✅ Face extraction pipeline
✅ Modular CV architecture
✅ Placeholder emotion prediction (to be upgraded)

🧠 System Architecture
Webcam → Face Detection → Face Extraction → Emotion Module → Output

🧱 Tech Stack
	•	Python
	•	OpenCV
	•	MediaPipe
	•	NumPy
	•	Streamlit (planned)
	•	HuggingFace (planned)
